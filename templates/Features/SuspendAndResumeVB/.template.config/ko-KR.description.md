사용자는 언제든지 앱을 일시 중단할 수 있으므로 이에 따라 앱을 준비해야 합니다. 이는 사용자가 앱을 떠나서 다시 돌아올 때 앱을 이전 상태로 되돌려 놓는 간단한 서비스입니다.
